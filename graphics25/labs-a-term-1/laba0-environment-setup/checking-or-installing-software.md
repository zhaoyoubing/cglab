# Checking or Installing Software

## OpenGL

GLEW and GLAD:  handle OpenGL function pointers.&#x20;

creates and setups Window and OpenGL context

helps in querying and loading OpenGL extensions

GLFW: &#x20;

an Open Source, multi-platform library in C for OpenGL, OpenGL ES and Vulkan development on the desktop. Provides a simple API for creating windows, contexts and surfaces, receiving input and events.

Supports Windows, macOS, Wayland and X11.

cross platform lib for management of windows and mouse/keyboard input.
